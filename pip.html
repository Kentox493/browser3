⚡ Attack Strategy

🎭 C. Fullscreen PiP Takeover PiP window dibuat sangat besar menutupi hampir seluruh layar. User tidak sadar ini PiP — terlihat seperti halaman biasa. Opener disembunyikan.
⚡ D. Race Condition + beforeunload Hold Teknik race condition asli: worker flood + navigate. Ditambah beforeunload dialog yang MENAHAN navigation — PiP hidup selama dialog tampil.
